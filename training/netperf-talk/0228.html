Hi,


I made a netperf-2.1pl3 patch for IPv6.


I tested on following environments.


     - FreeBSD 3.1 + KAME-snap-19990517 (WIDE Project, Japan)
     - Vine Linux (based on Red Hat 5.2 , Japanese Distribution)
          + kernel 2.2.9 + inet6-apps-0.35 + net-tools-1.52


Main changes are 
     * made "makefile.ipv6"
     * establish_control() in netlib.c
        - use "getaddrinfo" instead of "gethostbyname"
     * set_up_server() in netserver.c
        - use "getaddrinfo" instead of "gethostbyname"
     * nettest_ipv6.c and nettest_ipv6.h
        -  corrected some mistake


Thanks.
