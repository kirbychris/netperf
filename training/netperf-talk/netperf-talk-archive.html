The netperf-talk mailing list and archive are places where users of netperf can turn for expertise and experience in the care, feeding and running of the netperf benchmark.

In providing the resources of this list and archive, Hewlett-Packard is merely trying to help the network benchmarking community. It also helps keep one of their engineers happy . Please read the disclaimer.

The netperf-talk mailing list is maintained by majordomo. To subscribe or unsubscribe from the list, please send a message to majordomo@netperf.org. If you do not know how to use majordomo send a message with just "help" in the body to majordomo@netperf.org.

If you would like to send a message to the list, you should send mail to netperf-talk@netperf.org.

Please DO NOT send subscription requests to netperf-talk.

If you run into any problems with the netperf services provided by this server, please try using the Netperf Feedback page, or send me email.

Happy Benchmarking!
